AgentCockpit.app

Control Claude Code, Codex CLI, and OpenCode from one dashboard.

AgentCockpit is a zero-knowledge relay for AI coding sessions. You can move fast with autonomous agents while keeping final control of sensitive actions.

Zero-knowledge relaying Fast approval loop Works with your current tools

What It Is

A control plane for AI coding sessions across your machines.

Why It Is Secure

Session data is end-to-end encrypted. The relay routes traffic but cannot read payloads.

Why It Is Fast

Approvals, sessions, and terminal activity stream live in one place.

Use Case Walkthrough

How It Works
1. Agent sends approval requests and PTY output through AgentCockpit.
2. You review and approve or deny in the browser dashboard.
3. Session continues instantly with your decision.
Security Details
Crypto: ECDH P-256 + HKDF-SHA-256 + AES-256-GCM.
Relay can see: routing metadata (session/user IDs, timestamps, status).
Relay cannot see: encrypted tool payloads, stdin, and terminal output.
